# PICK Chart — Матрица приоритизации воздействия

**Автор:** Методология Lean Six Sigma
**Тип:** Приоритизация инициатив по эффекту и сложности
**Сложность:** ★☆☆☆☆ (простейшая)


## 📖 ИСТОРИЯ СОЗДАНИЯ

Методология Lean Six Sigma. PICK Chart родилась из потребности БЫСТРО фильтровать идеи после мозгового штурма — без сложных расчётов.

Possible, Implement, Challenge, Kill — четыре судьбы для каждой идеи. PICK проще, чем RICE, быстрее, чем Impact-Effort. Идеальный инструмент для ежеквартальных сессий улучшений.

## СУТЬ МЕТОДА
Матрица PICK оценивает инициативы по двум осям: Payoff (отдача) и Difficulty (сложность). 4 категории:

- **P — Possible (Возможно):** низкая отдача + лёгкая реализация. Делать во вторую очередь.
- **I — Implement (Внедрять):** высокая отдача + лёгкая реализация. ДЕЛАТЬ ПРЯМО СЕЙЧАС.
- **C — Challenge (Вызов):** высокая отдача + сложная реализация. Планировать, выделять ресурсы.
- **K — Kill (Убить):** низкая отдача + сложная реализация. УДАЛИТЬ без сожаления.

## АЛГОРИТМ

### Шаг 1. Выпиши все инициативы
Без цензуры — все идеи по улучшению.

### Шаг 2. Оцени каждую по двум осям (1-5)
- Payoff: насколько улучшит ключевые метрики?
- Difficulty: сколько усилий/времени/денег потребуется?

### Шаг 3. Распредели по квадрантам
Заполни матрицу 2×2.
