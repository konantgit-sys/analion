# Морфологический анализ (Morphological Analysis)

**Автор:** Фриц Цвикки (1940-е, астрофизика)
**Тип:** Структурный комбинаторный анализ
**Что анализирует:** Все возможные комбинации параметров системы


## 📖 ИСТОРИЯ СОЗДАНИЯ

1942 год. Швейцарский астрофизик Фриц Цвикки работает в Калифорнийском технологическом институте. Изучая реактивные двигатели, он сталкивается с проблемой: как систематически перебрать ВСЕ возможные конструкции?

Цвикки создаёт «морфологический ящик» — таблицу, где каждая ось = параметр системы, а ячейки = возможные значения. Перебор ВСЕХ комбинаций открывает решения, которые интуиция пропускает.

Метод быстро находит применение далеко за пределами астрофизики — от дизайна автомобилей до военной аналитики RAND Corporation.

## 👤 ОБ АВТОРЕ

**Фриц Цвикки** (1898–1974) — швейцарско-американский астрофизик. Открыл тёмную материю (1933). Известен резким характером — коллег называл «сферическими идиотами» (идиоты со всех сторон).

## 🌍 КОНТЕКСТ ПРИМЕНЕНИЯ

- Дизайн продуктов (перебор вариантов конструкции)
- Маркетинг (все возможные комбинации «продукт×канал×аудитория»)
- Военное планирование

## 🏭 ИЗВЕСТНЫЕ КЕЙСЫ

**RAND Corporation (1950-е):** Морфологический анализ сценариев ядерной войны — выявил сценарии, не замеченные традиционной аналитикой.

---

## Алгоритм анализа

### Шаг 1: Определи объект анализа
Что исследуем? Опиши систему одной фразой.

### Шаг 2: Выдели ключевые параметры (оси)
Какие независимые измерения есть у системы? 5-10 осей.
Примеры: размер, материал, форма, интерфейс, способ оплаты, 
целевая аудитория, канал доставки.

### Шаг 3: Для каждой оси — список значений
Для каждого параметра выпиши 3-7 возможных вариантов.
Не фильтруй — включай даже абсурдные.

### Шаг 4: Построй морфологический ящик (матрицу)
Создай таблицу: строки = параметры, столбцы = варианты.
Общее число комбинаций = произведение количества вариантов по всем осям.

### Шаг 5: Оцени комбинации
Для каждой комбинации оцени:
- Реализуемость (1-5)
- Новизна (1-5)
- Потенциальная ценность (1-5)

### Шаг 6: Выбери топ-5 комбинаций
Отбери наиболее перспективные комбинации для прототипирования.

### Шаг 7: Проверь на противоречия
Есть ли в выбранных комбинациях взаимоисключающие параметры?
Если да — передай в ТРИЗ для разрешения противоречий.

## Формат вывода
- **Объект:** [название]
- **Оси:** [список параметров с вариантами]
- **Матрица:** [таблица M×N]
- **Всего комбинаций:** [число]
- **Топ-5:** [список лучших]
- **Следующий шаг:** [ТРИЗ / прототип / A/B-тест]
